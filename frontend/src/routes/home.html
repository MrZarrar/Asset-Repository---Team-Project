Background

The goal of the project is to develop a Web-based repository system that supports the storage and access of different source code related assets of a software company. Examples of source code related assets could include:

  • Jar files
  • Shared libraries e.g. a security library to authenticate users through a service like LDAP
  • Scripts
  • Images
  • Information about the licensing of a particular asset
  • Asset usage information
  • Documentation about the asset
